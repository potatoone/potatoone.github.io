# 什么是MK One？

MK One 是一款轻量级文档管理工具，专为个人和团队的本地文档整理、阅读设计。它支持Markdown格式文档的解析、导航和存储，适合用于技术笔记、教程手册、知识库管理等场景。

## 核心特点
- **动态导航**：自动识别文档目录结构，生成层级侧边栏
- **即时渲染**：Markdown内容实时转换为美观的HTML
- **状态记忆**：自动保存阅读进度，刷新后保留当前页面
- **响应式设计**：适配桌面和移动设备，支持侧边栏折叠/展开

## 适用人群
- 开发者：整理技术笔记、API文档
- 创作者：编写教程、手册
- 学习者：管理学习资料、读书笔记

## 如何开始？
将你的Markdown文档按目录结构放入`/docs`文件夹，启动应用后即可在侧边栏看到自动生成的导航。
