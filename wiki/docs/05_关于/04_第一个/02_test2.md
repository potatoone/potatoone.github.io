# 进阶：自定义MK One主题颜色

MK One支持修改主题颜色以适配个人偏好，除了内置的预设颜色，还可通过简单配置自定义颜色。

## 基础主题切换
1. 点击顶部工具栏的“主题”按钮（格子图标）
2. 在弹出的面板中：
   - 勾选“夜间模式”切换明暗显示
   - 选择“主题颜色”预设（如蓝色、绿色、紫色）
   - 开启“AUTO”可跟随系统明暗模式自动切换

## 自定义主题颜色（进阶）
如果预设颜色不符合需求，可通过修改CSS文件自定义：

1. 找到项目中的`/css/index.css`文件
2. 搜索并修改以下颜色变量（示例）：
   ```css
   :root {
     --primary-color: #0e70c0; /* 主色调（默认蓝色） */
     --hover-color: #3a96dd;   /*  hover时的颜色 */
     --active-color: #ff6347;  /* 选中项颜色 */
     --text-color: #333;       /* 文本颜色 */
     --bg-color: #fff;         /* 背景颜色 */
   }
   
   /* 夜间模式颜色 */
   .dark-mode {
     --text-color: #ddd;
     --bg-color: #222;
   }
